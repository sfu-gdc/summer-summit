# Typography

Three font families, sized with UnoCSS's `text-*` scale (Wind4). Use `font-body`,
`font-header`, or `font-art` for family and `text-xs … text-9xl` for size. The sizes
below are read directly from that scale.

export const sizes = Object.values(textScale).map((t) => t.fontSize);
export const sample = 'The quick brown fox jumps over the lazy dog';

## UI and Prose

**Font:** {fonts.body}

**Weights:** 100, 300, 400, 600, 800 (normal and italic)

<Typeset
	fontSizes={sizes}
	fontWeight={400}
	sampleText={sample}
	fontFamily={`'${fonts.body}', sans-serif`}
/>
